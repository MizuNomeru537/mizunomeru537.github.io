카드형 환율 계산기

기존 은행 화면 느낌은 살리되 그대로 따라하지 않고 새로 구성한 버전이다. 통화별 국가 이미지를 넣었고, 자동 환율 조회와 직접 환율 입력을 둘 다 지원한다. 장중 시세와 차이가 날 때는 직접 환율 입력으로 바로 맞춰서 계산할 수 있다.

국가 이미지 포함 자동 환율 + 직접 입력 최근 추이 차트 통화 스왑
자동 환율 데이터 오픈 환율 API + 타임스탬프 표시
빠른 사용 흐름 통화 선택 → 금액 입력 → 계산
특징 GitHub Pages용 정적 페이지 구성
보유 금액
KRW
자동 환율은 외부 오픈 API의 최신 스냅샷을 사용한다. 장중 값과 다르면 직접 환율 입력으로 바로 맞출 수 있다.
환전 금액
USD
결과 금액은 일반 환산값이다. 은행별 우대율, 수수료, 스프레드는 포함하지 않았다. 실제 거래용 숫자가 필요하면 은행 고시나 증권사 시세를 한 번 더 확인하는 편이 안전하다.
