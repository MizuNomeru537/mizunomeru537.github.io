UUID 생성기

브라우저의 crypto.randomUUID()를 우선 사용하고, 지원되지 않는 경우에는 난수 기반 대체 생성기로 UUID v4 형식 문자열을 만듭니다.









한 번에 여러 개
최대 100개까지 한 번에 생성할 수 있습니다.


복사 지원
생성 결과 전체를 줄바꿈 상태 그대로 복사합니다.


실사용 가능
개발 테스트, 샘플 데이터, 식별자 발급용으로 바로 사용할 수 있습니다.
