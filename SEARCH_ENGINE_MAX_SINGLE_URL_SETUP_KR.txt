업로드 후 할 일:
1. 기존 파일 덮어쓰기

3. Search Console에 sitemap.xml 제출
4. 홈(/), 전체도구(/tools/), 대표 툴 3~5개 URL 색인 요청
5. 사이트 내부에서 새 페이지 링크가 모두 열리는지 확인
