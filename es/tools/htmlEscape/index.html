About this HTML Escape / Unescape

This HTML escape tool converts reserved characters into entities and restores them back when you need readable output again.

How to use

  1. Paste the text or HTML snippet.
  2. Choose Escape or Unescape.
  3. Copy the converted output for safe embedding or review.

FAQ

When should I escape HTML?

Escape HTML when you need to show raw tags or special characters safely inside documents, templates, or code samples.

What does unescape do?

It converts HTML entities like &amp;lt; back into normal characters like <.
