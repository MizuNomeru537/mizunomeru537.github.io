This page decodes JWT content but does not verify whether the signature is valid.

About this JWT Decoder

Use it to inspect JWT header fields, payload claims, expiration values, and token structure without installing anything.

How to use

  1. Paste a JWT token.
  2. Run Decode to read the header and payload.
  3. Review claims such as exp, iat, aud, iss, and sub.

Example

Useful while debugging APIs, auth flows, bearer tokens, and frontend sessions.

FAQ

Does this validate the signature?

No. It only decodes and displays the token parts.

Does my token leave the browser?

No. The token is processed on the page.
