AdSense 전체 보완본 안내

적용 내용
1. 모든 서비스용 HTML 페이지에 AdSense 스크립트 통일
2. google-adsense-account 메타 태그 추가
3. tools/exchangeCalculator/index.html 누락 수정
4. 홈 화면의 광고 자리 안내 문구 제거

중요
- 이 보완본은 Auto Ads 기준으로 바로 배포 가능한 형태입니다.
- 수동 광고(배너/인피드)를 넣으려면 AdSense에서 생성한 data-ad-slot 값이 추가로 필요합니다.
- ads.txt 는 기존 값이 유지됩니다.

배포 방법
- 압축 해제 후 파일 전체를 GitHub Pages 저장소 루트에 덮어쓰기 업로드
- 배포 후 브라우저 강력 새로고침(Ctrl+F5)
- AdSense 대시보드에서 Auto Ads가 켜져 있는지 확인
