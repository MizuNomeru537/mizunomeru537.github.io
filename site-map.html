Developer tools

Formatting, preview, hashing, and regex utilities for code-heavy workflows.

  • Code Minifier — Minify HTML, CSS, JavaScript, and JSON online to reduce size and remove unnecessary whitespace.
  • Hash Generator — Generate SHA-256 and SHA-1 hashes online in your browser for quick testing and verification work.
  • Regex Tester — Test regular expressions online with live match results, flags, and quick browser-based feedback.
  • HTML Preview — Write HTML and preview the rendered result side by side online without leaving the browser.

JSON, SQL, and CSV utilities for structured data cleanup and inspection.

  • JSON Formatter — Pretty print, validate, and clean JSON online with fast browser-based formatting and copy tools.

Text and writing tools

Cleanup, counting, diffing, placeholder text, and character helpers for daily content work.

  • Document Diff Checker — Compare two text blocks line by line online and highlight changes, removals, and additions.
  • Case Converter — Convert text into uppercase, lowercase, camelCase, snake_case, kebab-case, and title case online.
  • Word Counter — Count words, characters, lines, and bytes online with instant browser-based text analysis.
  • Lorem Ipsum Generator — Generate placeholder paragraphs online for UI mockups, CMS drafts, design previews, and layout testing.

Conversion tools

Encoding, timestamp, HTML entity, and color conversions for browser-based tasks.

  • Base64 Encode / Decode — Encode text to Base64 or decode Base64 strings online with instant browser-based conversion.
  • HTML Escape / Unescape — Escape HTML special characters to entities or unescape them back to readable text in your browser.
  • URL Encoder / Decoder — Encode or decode URLs and query strings online for safe sharing, debugging, and parameter handling.
  • Timestamp Converter — Convert Unix timestamps to readable date strings and turn readable dates back into timestamps online.
  • Color Converter — Convert HEX, RGB, and HSL color values online with a live preview for design and front-end work.
