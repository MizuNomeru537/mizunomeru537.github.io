기본 구조
- 영어 기본: /
- 한국어: /ko/
- 일본어: /ja/
- 중국어 간체: /zh-cn/
- 중국어 번체: /zh-tw/
- 프랑스어: /fr/
- 독일어: /de/
- 러시아어: /ru/
- 스페인어: /es/
- 라틴아메리카 스페인어: /es-419/
- 인도네시아어: /id/
- 아랍어: /ar/

적용 내용
- 각 페이지에 hreflang / x-default / self-canonical 반영
- 각 언어별 페이지 URL 생성
- sitemap.xml 전체 갱신
- 언어 탭 클릭 시 같은 페이지의 언어별 URL로 실제 이동

배포 후
1. 전체 업로드
2. Search Console 에서 sitemap.xml 재제출
3. /ko/ /ja/ /ko/tools/sqlFormatter/ 정도 대표 URL 수집 요청
