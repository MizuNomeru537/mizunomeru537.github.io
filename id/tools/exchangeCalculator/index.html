이 버전에서 바뀐 점

이전처럼 일일 기준값만 보고 실시간처럼 오해하기 쉬운 구성을 줄이고, 소스 성격을 구분해서 보이도록 다시 정리했습니다. 메인 ToolLab의 밝은 카드형 레이아웃과 맞추어 All 디자인도 통일했습니다.

사용 방법

  1. 보유 통화와 환전 통화를 고릅니다.
  2. 시장 스냅샷, 기준 환율, 직접 입력 중 하나를 선택합니다.
  3. 금액을 넣고 계산하면 결과와 적용 환율, 기준 시각, 데이터 출처가 함께 표시됩니다.

추천 사용 방식

장중 느낌이 필요할 때 → 시장 스냅샷 안정적으로 계산할 때 → 기준 환율 내가 본 숫자와 맞출 때 → 직접 입력

FAQ

왜 어떤 날은 기준 환율과 시장 스냅샷 값이 다르게 보이나요?

기준 환율은 일일 기준 데이터라 장중 변동을 바로 반영하지 않습니다. 시장 스냅샷은 장중에 더 가깝지만 지연 시세일 수 있습니다.

1497처럼 전일 값 비슷하게 보이던 문제는 어떻게 바뀌었나요?

이제 소스를 분리해서 보여주고, 시장 스냅샷이 실패하면 분명하게 기준 환율로 전환되었다고 표시합니다.

정확히 내가 본 숫자와 맞추고 싶으면 어떻게 하나요?

직접 입력 모드에서 1 기준통화 = ? 환전통화 값을 넣으면 됩니다. 예를 들어 1 USD = 1473 KRW처럼 바로 맞출 수 있습니다.
